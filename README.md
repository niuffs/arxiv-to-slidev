# arxiv-to-slidev

从 arXiv 论文 LaTeX 源码自动生成 Slidev 讨论班演示文稿的 Claude Code Skill。

> 对应英文版：`arxiv-to-slidev-en`

## 目录

```
├── Skills/
│   ├── arxiv-to-slidev-cn/       # 中文版 Skill
│   └── arxiv-to-slidev-en/       # 英文版 Skill
├── examples/
│   ├── cn/                       # 中文版示例 PPT
│   │   ├── slides.md
│   │   ├── figures/
│   │   └── styles/
│   └── en/                       # 英文版示例 PPT
│       ├── slides.md
│       ├── figures/
│       └── styles/
└── .github/workflows/
    └── deploy.yml               # GitHub Pages 自动部署
```

## 示例

- [中文版讨论班 PPT](https://niuffs.github.io/arxiv-to-slidev/examples/cn/)
- [英文版 Seminar PPT](https://niuffs.github.io/arxiv-to-slidev/examples/en/)

## 安装

将 `Skills/arxiv-to-slidev-cn/` 或 `Skills/arxiv-to-slidev-en/` 放入项目的 `.claude/skills/` 目录。
